## 安装说明

安装前请先删除X:\World of Warcraft里面的Cache、Interface、WTF三个文件夹，把下载的压缩包里面的Interface文件夹整个复制到X:\World of Warcraft目录。（X为游戏盘符，请根据自己游戏安装目录选择）
